Tatacara Installasi 1 DBMS (mysql)


1. Download MySQL Installer

Buka browser kamu, cari “MySQL”, lalu masuk ke situs resminya di mysql.com. Pilih tab Downloads → klik MySQL Community (GPL) Downloads → pilih MySQL Installer for Windows. Pilih versi yang kamu mau dan klik Download.

2. Jalankan Installer

Buka folder download, klik dua kali file .exe-nya, dan izinkan installer dijalankan.

3. Pilih Tipe Instalasi

  • Developer Default lengkap dengan server dan tools.
  • Server Only hanya server.
  • Client Only hanya aplikasi klien.
  • Full semua fitur (disarankan).
  • Custom atur manual.

4. Tentukan Lokasi Instalasi

Pilih folder default atau ubah sesuai kebutuhan, lalu klik Next.

5. Mulai Instalasi

Klik Execute untuk menginstal semua fitur seperti server, workbench, shell, router, dan dokumentasi.

6. Konfigurasi Server

Gunakan pengaturan default, atau ubah port dan nama komputer sesuai kebutuhan.

7. Buat Password Root

Gunakan password kuat (kombinasi huruf besar, angka, dan simbol), lalu klik Next.

8. Konfigurasi Windows Service

Klik Execute untuk membuat layanan MySQL di Windows, lalu klik Finish.

9. Verifikasi Koneksi

Masukkan password root, klik Next, dan jalankan verifikasi koneksi. Kalau berhasil, klik Finish.

10. Selesai!

Centang opsi untuk membuka MySQL Workbench setelah instalasi selesai. Sekarang MySQL siap digunakan.
